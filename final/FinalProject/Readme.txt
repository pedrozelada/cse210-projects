The program allows you to establish good habits, in the drop-down menu you can select different types of habits such as improving your diet, exercising, learning new things, etc. The program gives you the option to choose a habit from a list, or to have it generated randomly, or you can create a habit yourself. Each habit has a duration of 12 weeks, each week you must follow up to record if you fulfilled it or not. You can show all the habits and the weeks that you fulfilled or not.
